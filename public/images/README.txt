Use
Place the folder in your Astro project, for example: public/images/icons/
Then reference an asset with: /images/icons/icon-contact-phone.webp

Technical
- Format: lossless WebP with transparency (RGBA)
- Resolution: 1024 × 1024 px for clean scaling on retina displays
- Style: social icons retain recognisable platform styling; contact/service icons are a consistent navy outline.

Source & licensing reference
The free-source research for selecting an icon-system style used Simple Icons (CC0 project) for social brand references and Heroicons (MIT) for web-interface icon conventions. These files are web-ready, original rendered assets built for this Long Hưng package; you should still follow the platform brand-use rules when linking to Facebook, Instagram, Zalo, or TikTok.
